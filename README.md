## 🎨 Design & Animation Features

### Premium Animations
- **Text Reveal** — Words clip-up on scroll with staggered timing
- **Magnetic Buttons** — Buttons subtly follow cursor on hover (desktop)
- **3D Tilt Cards** — Property cards respond to mouse with perspective transform
- **Parallax Layers** — Multi-depth scroll parallax on hero and images
- **Count-Up Numbers** — Stats animate from 0 when scrolled into view
- **Infinite Marquee** — Partner logos scroll continuously
- **Clip-Path Reveals** — Images reveal with cinematic crop animation
- **Grain Overlay** — Subtle noise texture for premium depth
- **Custom Cursor** — Dot + ring cursor with mix-blend-difference (desktop)
- **Orchestrated Preloader** — Branded loading sequence with progress bar

### UI/UX Details
- **Dark / Light Mode** — Toggle with localStorage persistence, no flash on load
- **Smooth Scroll** — Lenis-powered scroll synced with GSAP ticker
- **Responsive Design** — Mobile-first, search bar hidden on mobile
- **Hover Micro-interactions** — Every interactive element has refined feedback
- **Staggered Entrances** — Sections animate in with expo easing
- **Focus States** — Visible focus rings for keyboard navigation

## 📂 Project Structure

```
src/
├── app/
│   ├── layout.tsx          → Root layout (fonts, metadata, theme)
│   ├── page.tsx            → Home page composition
│   └── globals.css         → CSS variables, keyframes, utilities
├── components/
│   ├── sections/           → Page sections (Hero, Properties, etc.)
│   ├── layout/             → Navigation, Footer
│   ├── ui/                 → Reusable primitives (TiltCard, TextReveal, etc.)
│   └── effects/            → Visual effects (Cursor, Preloader, Grain)
├── hooks/                  → Custom hooks (useGsap, useMagnetic)
├── context/                → Theme provider (dark/light)
└── lib/                    → GSAP init, font config
```

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/xsid-69/RealEsate-LandingPage.git

# Navigate to project
cd RealEsate-LandingPage

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
